TestStruct Struct

A test struct having everything that can be declared on a struct.

  • Generic parameters
  • Fields
  • Constants
  • Events
  • Properties
  • Indexer
  • Methods
  • Shadowed Methods
  • Attributes both on the type itself as well as on each type of member.
  • Nested types.
  • Inheritance scenarios (overrides and virtuals).


Implemented interfaces: ITestExtendedBaseInterface.
