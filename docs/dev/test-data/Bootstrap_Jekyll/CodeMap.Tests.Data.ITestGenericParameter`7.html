ITestGenericParameter Interface

This is an interface with a number of generic parameters, used as an example for documenting generic parameters and their constraints.

Generic Parameters

  • TParam1

    This generic parameter is covariant.

    No constraints, can be any type, is covariant.

  • TParam2

    This generic parameter is contravariant.

    This generic parameter is constrained to reference types.

    Must be a reference type, is contravariant.

  • TParam3

    This generic parameter is constrained to value types.

    Must be a value type (nullables do not count).

  • TParam4

    This generic parameter is constrained to having a public parameterless constructor.

    Must have a public parameterless constructor.

  • TParam5

    This generic parameter has the following type constraints: TParam1.

    Must be of same type as TParam1 or a subtype of TParam1.

  • TParam6

    This generic parameter has the following type constraints: TParam1, IComparable<TParam1>.

    Must be of same type as TParam1 or a subtype of TParam1 and implement IComparable<T> or be IComparable<T>.

  • TParam7

    This generic parameter is constrained to unamanged types.

    Must be an unmanaged type.

Example

For TParam6 the following configuration is valid, int implements IComparable<T>. 

ITestGenericParameter<int, object, int, object, int, int>
