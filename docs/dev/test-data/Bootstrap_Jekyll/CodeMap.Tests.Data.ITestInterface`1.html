ITestInterface Interface

A test interface having everything that can be declared on an interface.

  • Generic parameters
  • Events
  • Properties
  • Indexer
  • Methods
  • Shadowed Events
  • Shadowed Properties
  • Shadowed Methods
  • Attributes both on the type itself as well as on each type of member.

    A generic parameter, used for different things.
