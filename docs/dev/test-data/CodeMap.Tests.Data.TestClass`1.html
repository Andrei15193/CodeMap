TestClass Class

A test class having everything that can be declared on a class.

  • Generic parameters
  • Fields
  • Constants
  • Events
  • Properties
  • Indexer
  • Methods
  • Shadowed Fields
  • Shadowed Events
  • Shadowed Properties
  • Shadowed Methods
  • Attributes both on the type itself as well as on each type of member.
  • Nested types.
  • Inheritance scenarios (overrides, virtuals, sealed overrides).

Base type: TestBaseClass.
Implemented interfaces: ITestExtendedBaseInterface.
