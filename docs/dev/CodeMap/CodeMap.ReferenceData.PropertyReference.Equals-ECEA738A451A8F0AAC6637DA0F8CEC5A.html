Equals Method

Determines whether the current PropertyReference is equal to the provided assemblyName.

This is a sealed method.

This is a public method.

Parameters

  • assemblyName: AssemblyName

    The AssemblyName to compare to.

Returns: bool

Returns true if the current PropertyReference references the provided assemblyName; false otherwise.

Remarks

This method always returns false because an AssemblyName cannot represent a property reference.
