Hyperlink Method

Creates a HyperlinkDocumentationElement with the provided destination and text.

This is a static method.

This is a public method.

Parameters

  • destination: string

    The hyperlink destination (URL).

  • text: string

    The hyperlink text.

  • xmlAttributes: IReadOnlyDictionary<string, string>

    The XML attributes specified on the inline code element.

Returns: HyperlinkDocumentationElement

Returns a HyperlinkDocumentationElement with the provided text.

Exceptions

  • ArgumentNullException

    Thrown when destination or text are null.

  • ArgumentException

    Thrown when xmlAttributes contain null values.
