CreateNamespace Method

Creates a NamespaceReference for the provided namespace name and assemblyName.

This is a public method.

Parameters

  • name: string

    The name of the namespace.

  • assemblyName: AssemblyName

    The AssemblyName in which the namespace is declared.

Returns: NamespaceReference

Returns an AssemblyName for the provided namespace name and assemblyName.

Exceptions

  • ArgumentNullException

    Thrown when assemblyName is null.
