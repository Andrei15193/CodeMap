Equals Method

Determines whether the current MemberReference is equal to the provided obj.

This is a sealed method.

This is a public method.

Parameters

  • obj: object

    The object to compare to.

Returns: bool

Returns true if the current MemberReference references the provided obj; false otherwise.

Remarks

If the provided obj is a MemberInfo, Assembly or AssemblyName then the comparison is done by checking whether the current instance actually maps to the provided actual instance. Otherwise the equality is determined by comparing references.
