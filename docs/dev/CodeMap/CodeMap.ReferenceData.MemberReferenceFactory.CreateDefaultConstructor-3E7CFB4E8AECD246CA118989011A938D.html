CreateDefaultConstructor Method

Creates a ConstructorReference for the default constructor of the given type.

This is a public method.

Parameters

  • type: Type

    The Type for which to create a ConstructorReference, must be a struct (value type).

Returns: ConstructorReference

Returns a ConstructorReference for the default constructor of the given type.

Exceptions

  • ArgumentNullException

    Thrown when type is null.

  • ArgumentException

    Thrown when type is not a struct (value type).
