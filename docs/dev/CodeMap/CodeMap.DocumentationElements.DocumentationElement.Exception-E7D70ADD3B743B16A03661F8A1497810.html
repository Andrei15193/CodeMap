Exception Method

Creates an ExceptionDocumentationElement with the provided description.

This is a static method.

This is a public method.

Parameters

  • exception: MemberReferenceDocumentationElement

    The exception being thrown.

  • description: IEnumerable<BlockDocumentationElement>

    The content of the exception XML element.

Returns: ExceptionDocumentationElement

Returns an ExceptionDocumentationElement with the provided description.

Exceptions

  • ArgumentNullException

    Thrown when description is null.

  • ArgumentException

    Thrown when description contains null elements.
