MemberReference Method

Creates a MemberNameReferenceDocumentationElement with the provided canonicalName.

This is a static method.

This is a public method.

Parameters

  • canonicalName: string

    The canonical name for a member referred using a see XML element.

  • xmlAttributes: IReadOnlyDictionary<string, string>

    The XML attributes specified on the member reference element.

Returns: MemberNameReferenceDocumentationElement

Returns a MemberNameReferenceDocumentationElement with the provided canonicalName.

Exceptions

  • ArgumentNullException

    Thrown when canonicalName is null.

  • ArgumentException

    Thrown when xmlAttributes contain null values.
