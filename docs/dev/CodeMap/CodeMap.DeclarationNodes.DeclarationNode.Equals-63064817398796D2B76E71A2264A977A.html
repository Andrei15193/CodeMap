Equals Method

Determines whether the current DeclarationNode is equal to the provided assembly.

This is a public method.

Parameters

  • assembly: Assembly

    The Assembly to compare to.

Returns: bool

Returns true if the current DeclarationNode references the provided assembly; false otherwise.
