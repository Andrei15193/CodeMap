TryFindMemberInfoFor Method

Attempts to find a MemberInfo with the provided canonicalName.

This is a public method.

Parameters

  • canonicalName: string

    The canonical name of the member to search for.

Returns: MemberInfo

Returns the MemberInfo with the provided canonicalName if found; otherwise null.

Exceptions

  • ArgumentException

    Thrown when the canonical name is invalid.
