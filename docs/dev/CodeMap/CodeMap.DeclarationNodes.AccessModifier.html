AccessModifier Enum

Represents the access modifier of a declared member of an assembly.

Members

Name Value Summary
Private 0

The member is visible only to the declaring type.

FamilyAndAssembly 1

The member is visible only to the declaring assembly and only to subtypes declared by the same assembly.

Assembly 2

The member is visible only to the declaring assembly.

Family 3

The member is visible to all subtypes.

FamilyOrAssembly 4

The member is visible to the declaring assembly and to all subtypes declared by any assembly.

Public 5

The member is visible to all types regradless of the assembly they are declared by.
