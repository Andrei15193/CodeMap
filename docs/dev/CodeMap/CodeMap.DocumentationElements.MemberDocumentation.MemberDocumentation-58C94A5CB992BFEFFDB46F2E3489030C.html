MemberDocumentation Constructor

Initializes a new instance of the MemberDocumentation class.

This is a public constructor.

Parameters

  • canonicalName: string

    The canonical name of the documented member.

  • summary: SummaryDocumentationElement

    The summary section.

  • genericParameters: IReadOnlyDictionary<string, BlockDescriptionDocumentationElement>

    The generic parameters documentation.

  • parameters: IReadOnlyDictionary<string, BlockDescriptionDocumentationElement>

    The parameters documentation.

  • returns: BlockDescriptionDocumentationElement

    The returns section.

  • exceptions: IReadOnlyList<ExceptionDocumentationElement>

    The exceptions documentation.

  • remarks: RemarksDocumentationElement

    The remarks section.

  • examples: IEnumerable<ExampleDocumentationElement>

    The examples sections.

  • value: ValueDocumentationElement

    The value section.

  • relatedMembers: IEnumerable<MemberReferenceDocumentationElement>

    The related members.

Exceptions

  • ArgumentNullException

    Thrown when canonicalName is null.
