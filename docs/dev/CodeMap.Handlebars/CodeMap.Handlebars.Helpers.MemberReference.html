MemberReference Class

A helper used to generate hyperlinks for a DeclarationNode or a MemberReference.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Writes a hyperlink to the provided textWriter.

Example

The following template will generate a paragraph containing a hyperlink to the given parameter. 

<p>{{MemberReference declaringType}}</p>

If the current context exposes a declaringType property that is a ClassDeclaration named CustomType, the output will be as follows: 

<p><a href="CustomType.html">CustomType</a></p>
