Concat Class

A concatenation helper, concatenates each provided parameter.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Name Access Modifier Summary
Apply(TextWriter, object, object[]) public

Applies the concatenation by writing each parameter to the provided writer.

Example

The following template will generate a paragraph containing the concatenated values. 

<p>{{Concat 'Is this your ' this ' value?}}</p>

If the current context is a number, say 5 the output would be: 

<p>Is this your 5 value?</p>
