MemberAccessModifier Class

A helper used to get the access modifier textual description for a DeclarationNode. Most useful for properties where the access modifier can be changed for one of the accessors.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Example

The following template will generate a paragraph containing the access modifier of the given parameter. 

<p>{{MemberAccessModifier declaration}}</p>

If the current context exposes a declaration property that is a public property with a private setter PropertyDeclaration, the output will be as follows: 

<p>public get; private set</p>
