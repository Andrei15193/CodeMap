GetBlockHelpers Method

Gets a collection of block helpers that are available in each template.

This is a virtual method.

This is a protected method.

Parameters

  • memberReferenceResolver: IMemberReferenceResolver

    The IMemberReferenceResolver used to resolve MemberReferences.

Returns: IEnumerable<IHandlebarsBlockHelper>

Returns a collection of block helpers to use in templates.

Remarks

If there are multiple helpers that have the same name, only the latest will be registered in the Handlebars engine.
