GetHelpers Method

Gets a collection of inline helpers that are available in each template.

This is a virtual method.

This is a protected method.

Parameters

  • memberReferenceResolver: IMemberReferenceResolver

    The IMemberReferenceResolver used to resolve MemberReferences.

Returns: IEnumerable<IHandlebarsHelper>

Returns a collection of inline helpers to use in templates.

Remarks

If there are multiple helpers that have the same name, only the latest will be registered in the Handlebars engine. This enables easy overriding of helpers as such: 

 public class MyCustomHandlebarsTemplateWriter : HandlebarsTemplateWriter
 {
     public MyCustomHandlebarsTemplateWriter(IMemberReferenceResolver memberReferenceResolver)
         : base(memberReferenceResolver)
     {
     }

     protected override IEnumerable<IHandlebarsHelper> GetHelpers(IMemberReferenceResolver memberReferenceResolver)
     {
         foreach (var helper in base.GetHelpers(memberReferenceResolver))
             yield return helper;

         yield return new MyCustomSemver();
     }
 }

If MyCustomSemver has the same helper name as Semver then MyCustomSemver will override the Semver helper.
