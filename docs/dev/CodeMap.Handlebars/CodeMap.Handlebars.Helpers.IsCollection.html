IsCollection Class

A helper used to determine whether a value is a collection.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Constructors

IsCollection() public

Properties

Name Access Modifier Summary
Name public get

Gets the name of the helper.

Methods

Name Access Modifier Summary
Apply(TextWriter, object, object[]) public

Writes true to the provided writer if the first parameter (when provided) is an IEnumerable. When there are no parameters provided then the current context is checked whether it is an IEnumerable.

Example

The following template will generate a paragraph if the provided values is a collection. 

{{#if (IsCollection this)}}<p>A collection</p>{{/if}}
