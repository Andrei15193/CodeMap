Apply Method

Writes the access modifier to the provided writer for the first parameter or current context.

This is a public method.

Parameters

  • writer: TextWriter

    The TextWriter to write the result to.

  • context: object

    The context in which this helper is called.

  • parameters: object[]

    The parameter with which this helper has been called.

Exceptions

  • ArgumentException

    Thrown when the first parameter is not a DeclarationNode or when not provided and the given context is not a DeclarationNode.
