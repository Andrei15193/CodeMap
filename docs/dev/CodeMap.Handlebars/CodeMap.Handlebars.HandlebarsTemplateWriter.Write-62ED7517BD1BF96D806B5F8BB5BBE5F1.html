Write Method

Writes the provided templateName with in the given context to the provided textWriter.

This is a public method.

Parameters

  • textWriter: TextWriter

    The TextWriter to which to write the template.

  • templateName: string

    The name of the template to apply.

  • context: object

    The context in which to apply the template.
