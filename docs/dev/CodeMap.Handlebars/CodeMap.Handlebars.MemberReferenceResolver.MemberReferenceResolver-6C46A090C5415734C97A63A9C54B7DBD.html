MemberReferenceResolver Constructor

Initializes a new instance of the MemberReferenceResolver class.

This is a public constructor.

Parameters

  • memberReferenceResolvers: IReadOnlyDictionary<Assembly, IMemberReferenceResolver>

    A collection of Assemblys with associated IMemberReferenceResolvers used for determining which specfic IMemberReferenceResolver.

  • defaultMemberReferenceResolver: IMemberReferenceResolver

    A default IMemberReferenceResolver to use in case no IMemberReferenceResolver matches an Assembly.

Exceptions

  • ArgumentException

    Thrown when memberReferenceResolvers is null or contains null values.
