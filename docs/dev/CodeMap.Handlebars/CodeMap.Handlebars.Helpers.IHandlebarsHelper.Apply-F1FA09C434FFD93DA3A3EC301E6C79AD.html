Apply Method

Applies the helper in the provided context with the given parameters.

This is a public method.

Parameters

  • writer: TextWriter

    The TextWriter to which to write the result.

  • context: object

    The context in which the helper has been called.

  • parameters: object[]

    The parameters with which the helper was called.

Exceptions

  • ArgumentException

    Thrown when any of the provided parameters are invaid.
