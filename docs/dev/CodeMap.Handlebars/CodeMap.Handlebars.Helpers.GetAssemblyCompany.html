GetAssemblyCompany Class

A helper used to get the company name from an AssemblyDeclaration.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Gets the assembly company attribute value from the first parameter or current context.

Example

The following template will generate a paragraph containing the company name. 

<p>{{GetAssemblyCompany}}</p>

If the current context is an AssemblyDeclaration with the company name set to MyAwesomeCompany, the output would be: 

<p>MyAwesomeCompany</p>
