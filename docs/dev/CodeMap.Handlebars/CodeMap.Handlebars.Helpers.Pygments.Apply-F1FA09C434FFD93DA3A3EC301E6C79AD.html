Apply Method

Writes a code block with highlighting to the provided writer.

This is a public method.

Parameters

  • writer: TextWriter

    The TextWriter to write the result to.

  • context: object

    The context in which this helper is called.

  • parameters: object[]

    The parameter with which this helper has been called.

Remarks

The first parameter is the code to be highlighted while the second specifies the language for the lexer. If no language is specified or a lexer is not defined for that language then no highlighting is made. The currently supported languages are as follows:

  • C#
  • XML - Extensible Markup Language
  • HTML - HyperText Markup Language
  • CSS - Cascading Style Sheets
  • JS - JavaScript
  • TS - TypeScript
  • Bash - Bourne Again Shell (Linux Bash)
  • SQL - Structured Query Language
