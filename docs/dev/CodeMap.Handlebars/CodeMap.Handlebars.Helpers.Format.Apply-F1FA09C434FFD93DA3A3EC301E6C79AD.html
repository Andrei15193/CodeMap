Apply Method

Formats the first parameter which is expected to be an IFormattable by applying the format string provided through the second parameter which is expected to be a string.

This is a public method.

Parameters

  • writer: TextWriter

    The TextWriter to write the result to.

  • context: object

    The context in which this helper is called.

  • parameters: object[]

    The parameter with which this helper has been called.

Exceptions

  • ArgumentException

    Thrown when the first parameter does not implement IFormattable or when the second parameter is not a string.
